# Examples

Worked examples of common UI patterns built with `@studiometa/js-toolkit`. Each example shows the HTML, the component code, and the app setup.

## Basics

- [Counter component](/examples/counter-component/) — Increment and decrement a number with refs and event handling
- [Accordion](/examples/accordion/) — Show and hide a content section with ARIA support

## Patterns

- [Tabs](/examples/tabs/) — Switch between panels with keyboard navigation and ARIA roles
- [Modal](/examples/modal/) — Open a dialog with a backdrop, focus trapping, and Escape to close
- [Form validation](/examples/form-validation/) — Validate fields in real time with error messages

## Advanced

- [Scroll-linked animation](/examples/scroll-linked-animation/) — Drive parallax and skew effects from scroll position
- [Lazy image](/examples/lazy-image/) — Defer image loading with the `withMountWhenInView` decorator
- [Teleport refs](/examples/teleport-refs/) — Move DOM elements outside the component tree while keeping ref access
