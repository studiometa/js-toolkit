# withoutTrailingSlash

## Usage

```js twoslash
import { withoutTrailingSlash } from '@studiometa/js-toolkit/utils';

withoutTrailingSlash('string/'); // "string"
withoutTrailingSlash('string'); // "string"
```

### Parameters

- `string` (`string`): The string to modify.

### Return value

- `string`: The modified string.
