# Lifecycle hooks

## `mounted`

Called after the instance has been mounted.

**Arguments**

> This method has no argument.

**Example**

```js {9-12} twoslash
import { Base } from '@studiometa/js-toolkit';

export default class Component extends Base {
  static config = {
    name: 'Component',
    log: true,
  };

  mounted() {
    // Logs 'mounted' when the component is mounted
    this.$log('mounted');
  }
}
```
