# Events

## Lifecycle events

### `mounted`

Emitted when the instance is mounted.

```js
import { Base } from '@studiometa/js-toolkit';

class App extends Base {
  static config = {
    name: 'App',
  };
}

const app = new App(document.body);

app.$on('mounted', () => {
  console.log('App is mounted');
});
```
